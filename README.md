# AIQuiz / AI問題集メーカー

教材（教科書・参考書の画像／PDF／貼り付けテキスト）を読み込ませると、**Claude API** が問題を自動生成し、出題・採点・復習・成績管理までしてくれる学習アプリです。**1つのHTMLファイルで完結**しています（フレームワーク不使用・素のJS）。

> ⚠️ 個人制作の学習補助ツールです。Anthropic 社をはじめとする企業の公式製品ではありません。AIが生成する問題・解説・図・採点は**誤りを含むことがあります**。重要な学習では必ず元の教材・一次資料で確認してください。

---

## 主な機能

- **多彩な出題形式**：選択式／仕分け表／並べ替え／穴埋め／表穴埋め／記述（AI採点）
- **数式表示**：KaTeX を同梱（完全オフラインで数式描画）。数式キーボード・テンプレートビルダー付き
- **賢い採点**：表記揺れの正規化＋AIによる別解（同値）判定＋手動で「別解として正解にする」
- **適応難易度**：直近の正誤に応じて出題難易度を自動調整
- **図・出典**：AIによる作図／教材内の図・表の切り出し表示／出典ページの参照
- **学習履歴・苦手復習（SRS）**：成績・トピック別理解度の記録、間違えた問題の後日再出題
- **保存・印刷**：問題セットをJSONで保存（キー不要で再生可）、結果画面から印刷
- **スマホ対応**：レイアウト・タッチ操作・数式キーボード・カスタムUIを最適化済み

## 使い方

1. このアプリを**ブラウザで開く**（Chrome / Edge / Brave 推奨）
2. [Anthropic Console](https://console.anthropic.com/settings/keys) で **APIキー**（`sk-ant-…`）を発行し、[Billing](https://console.anthropic.com/settings/billing) でクレジットを購入
3. アプリの「APIキー」欄に貼り付ける（このブラウザ内 localStorage にのみ保存。次回から不要）
4. 教材（PDF・画像）をアップロード、または本文をテキスト欄に貼り付け →「**AIで問題を生成する**」

- **お試しモード（キー不要）**：APIキーが無くてもサンプル問題を体験できます。
- **保存した問題セット（JSON）の再生**：キー不要で再出題できます。

### 料金について
AI利用料金は**従量課金**で、**すべて利用者自身の Anthropic アカウントに課金**されます。目安は1回の生成あたり数円〜十数円程度（教材サイズ・問題数で変動）。記述のAI採点や別解判定などで追加の呼び出しが発生することがあります。**共有PCではAPIキーを入力しないでください。**

## 動かし方

### A. リンクで使う（ホスティング・推奨）
https で配信すると、リンクを開くだけで使え、**ホーム画面への追加（PWA）も有効**になります。
GitHub Pages を使う場合：リポジトリの **Settings → Pages** で `main` ブランチを公開 → `https://<ユーザー名>.github.io/AIQuiz-app/ai_quiz_app_v5.html` で開けます。

### B. ローカルファイルで使う
`ai_quiz_app_v5.html` を直接ブラウザで開きます。一部ブラウザでは `file://` だとAPIがブロックされることがあるため、同梱の起動スクリプトで `http://localhost` 経由を推奨します。
- **Windows**：`起動.bat` をダブルクリック（`server.ps1` を使ったローカルサーバ）
- **Mac / Linux**：`起動.bat` をターミナルから実行、または `.command` にリネーム

> ローカルファイル（`file://` / `content://`）では**ホーム画面追加・PWAインストールは効きません**（安定したURLが無いため）。アプリ化したい場合は A のホスティングを使ってください。

## プライバシー

- **APIキー**はこのファイルには保存されず、お使いの**ブラウザ内（localStorage）にのみ**保存されます。Anthropic への直接通信以外に送信されません。
- アップロードした**教材・テキストは問題生成のため Anthropic の API へ送信**されます。本アプリは独自サーバーを持たず、データの収集・保存・第三者提供はしません。
- 個人情報・機密情報・未公開の試験問題などは**アップロードしないでください**。
- 学習履歴・設定はブラウザ単位の保存で、端末間では共有されません。

詳細な利用条件・免責事項は、アプリ内の「**利用上の注意**」（全18条）をご覧ください。
